# coffee-shop-crud-python
A Django-based Coffee Shop CRUD application built inside a Python virtual environment (venv). Includes full Create, Read, Update, and Delete functionality and uses MongoDB as the main database for managing products and shop data.

**Coffee Shop CRUD Application built with Django, featuring full CRUD (Create, Read, Update, Delete) functionality for managing coffee shop products and operations.
The project is developed inside a virtual environment (venv) and uses MongoDB as the primary database for storing application data.

Features:

Full CRUD operations for products (coffee items, categories, etc.)

Backend built with Django

MongoDB integration for data storage

Developed and managed within a Python virtual environment (venv)

Clean and simple project structure**
